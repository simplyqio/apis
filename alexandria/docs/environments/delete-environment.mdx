title: Delete Environment
---

Once an environment has been delted it cannot be recovered. Please be sure you want to delete this environment before proceeding.

There must be one environment in the system. There you will not be able to delete the last environment.

1. Under the settings cog in the top right corner, select **Environments**.
1. You will be taken to the [currently selected environment settings page](https://app.simplyq.io/environment/settings).
1. Follow the red **Delete** button at the bottom of the page.
1. On completion of the request the latest environment will be selected and you will be redirected to the [applications dashboard](https://app.simplyq.io/applications).
